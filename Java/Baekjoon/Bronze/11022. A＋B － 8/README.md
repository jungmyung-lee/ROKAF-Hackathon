### Problem Description

<p>두 정수 A와 B를 Input받은 다음, A+B를 Output하는 프로그램을 작성하시오.</p>

### Input 

 <p>첫째 줄에 테스트 케이스의 개수 T가 주어진다.</p>

<p>각 테스트 케이스는 한 줄로 이루어져 있으며, 각 줄에 A와 B가 주어진다. (0 < A, B < 10)</p>

### Output 

 <p>각 테스트 케이스마다 "Case #x: A + B = C" 형식으로 Output한다. x는 테스트 케이스 번호이고 1부터 시작하며, C는 A+B이다.</p>
