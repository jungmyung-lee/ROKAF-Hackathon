### Problem Description

<p>영어 대소문자와 공백으로 이루어진 String이 주어진다. 이 String에는 몇 개의 단어가 있을까? 이를 구하는 프로그램을 작성하시오. 단, 한 단어가 여러 번 등장하면 등장한 횟수만큼 모두 세어야 한다.</p>

### Input 

 <p>첫 줄에 영어 대소문자와 공백으로 이루어진 String이 주어진다. 이 String의 길이는 1,000,000을 넘지 않는다. 단어는 공백 한 개로 구분되며, 공백이 연속해서 나오는 경우는 없다. 또한 String은 공백으로 시작하거나 끝날 수 있다.</p>

### Output 

 <p>첫째 줄에 단어의 개수를 Output한다.</p>
