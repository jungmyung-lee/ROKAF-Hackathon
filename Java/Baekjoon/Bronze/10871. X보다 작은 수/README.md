### Problem Description

<p>정수 N개로 이루어진 수열 A와 정수 X가 주어진다. 이때, A에서 X보다 작은 수를 모두 Output하는 프로그램을 작성하시오.</p>

### Input 

 <p>첫째 줄에 N과 X가 주어진다. (1 ≤ N, X ≤ 10,000)</p>

<p>둘째 줄에 수열 A를 이루는 정수 N개가 주어진다. 주어지는 정수는 모두 1보다 크거나 같고, 10,000보다 작거나 같은 정수이다.</p>

### Output 

 <p>X보다 작은 수를 Input받은 순서대로 공백으로 구분해 Output한다. X보다 작은 수는 적어도 하나 존재한다.</p>
