### Problem Description

<p>두 개의 자연수를 Input받아 최대 공약수와 최소 공배수를 Output하는 프로그램을 작성하시오.</p>

### Input 

 <p>첫째 줄에는 두 개의 자연수가 주어진다. 이 둘은 10,000이하의 자연수이며 사이에 한 칸의 공백이 주어진다.</p>

### Output 

 <p>첫째 줄에는 Input으로 주어진 두 수의 최대공약수를, 둘째 줄에는 Input으로 주어진 두 수의 최소 공배수를 Output한다.</p>
