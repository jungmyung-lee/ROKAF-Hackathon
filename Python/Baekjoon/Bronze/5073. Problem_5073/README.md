### Problem Description

<p>삼각형의 세 변의 길이가 주어질 때 변의 길이에 따라 다음과 같이 정의한다.</p>

<ul>
	<li>Equilateral :  세 변의 길이가 모두 같은 경우</li>
	<li>Isosceles : 두 변의 길이만 같은 경우</li>
	<li>Scalene : 세 변의 길이가 모두 다른 경우</li>
</ul>

<p>단 주어진 세 변의 길이가 삼각형의 조건을 만족하지 못하는 경우에는 "Invalid" 를 Output한다. 예를 들어 6, 3, 2가 이 경우에 해당한다. 가장 긴 변의 길이보다 나머지 두 변의 길이의 합이 길지 않으면 삼각형의 조건을 만족하지 못한다.</p>

<p>세 변의 길이가 주어질 때 위 정의에 따른 결과를 Output하시오.</p>

### Input 

 <p>각 줄에는 1,000을 넘지 않는 양의 정수 3개가 Input된다. 마지막 줄은 0 0 0이며 이 줄은 계산하지 않는다.</p>

### Output 

 <p>각 Input에 맞는 결과 (Equilateral, Isosceles, Scalene, Invalid) 를 Output하시오.</p>
