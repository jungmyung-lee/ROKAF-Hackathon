### Problem Description

<p>문장이 주어졌을 때, 단어를 모두 뒤집어서 Output하는 프로그램을 작성하시오. 단, 단어의 순서는 바꿀 수 없다. 단어는 영어 알파벳으로만 이루어져 있다.</p>

### Input 

 <p>첫째 줄에 테스트 케이스의 개수 T가 주어진다. 각 테스트 케이스는 한 줄로 이루어져 있으며, 문장이 하나 주어진다. 단어의 길이는 최대 20, 문장의 길이는 최대 1000이다. 단어와 단어 사이에는 공백이 하나 있다.</p>

### Output 

 <p>각 테스트 케이스에 대해서, Input으로 주어진 문장의 단어를 모두 뒤집어 Output한다.</p>
