### Problem Description

<p>두 정수 A와 B가 주어졌을 때, A와 B를 비교하는 프로그램을 작성하시오.</p>

### Input 

 <p>첫째 줄에 A와 B가 주어진다. A와 B는 공백 한 칸으로 구분되어져 있다.</p>

### Output 

 <p>첫째 줄에 다음 세 가지 중 하나를 Output한다.</p>

<ul>
	<li>A가 B보다 큰 경우에는 '<code>></code>'를 Output한다.</li>
	<li>A가 B보다 작은 경우에는 '<code><</code>'를 Output한다.</li>
	<li>A와 B가 같은 경우에는 '<code>==</code>'를 Output한다.</li>
</ul>
