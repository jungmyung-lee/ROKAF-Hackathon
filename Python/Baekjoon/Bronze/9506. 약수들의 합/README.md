### Problem Description

<p>어떤 숫자 n이 자신을 제외한 모든 약수들의 합과 같으면, <span style="line-height:1.6em">그 수를 완전수라고 한다. </span></p>

<p>예를 들어 6은 6 = 1 + 2 + 3 으로 완전수이다.</p>

<p>n이 완전수인지 아닌지 판단해주는 프로그램을 작성하라.</p>

### Input 

 <p>Input은 테스트 케이스마다 한 줄 간격으로 n이 주어진다. (2 < n < 100,000)</p>

<p>Input의 마지막엔 -1이 주어진다.</p>

### Output 

 <p>테스트케이스 마다 한줄에 하나씩 Output해야 한다.</p>

<p>n이 완전수라면, n을 n이 아닌 약수들의 합으로 나타내어 Output한다(Example Output 참고).</p>

<p>이때, 약수들은 오름차순으로 나열해야 한다.</p>

<p>n이 완전수가 아니라면 n is NOT perfect. 를 Output한다.</p>
