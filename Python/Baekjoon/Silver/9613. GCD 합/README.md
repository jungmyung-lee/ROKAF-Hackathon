### Problem Description

<p>양의 정수 n개가 주어졌을 때, 가능한 모든 쌍의 GCD의 합을 구하는 프로그램을 작성하시오.</p>

### Input 

 <p>첫째 줄에 테스트 케이스의 개수 t (1 ≤ t ≤ 100)이 주어진다. 각 테스트 케이스는 한 줄로 이루어져 있다. 각 테스트 케이스는 수의 개수 n (1 < n ≤ 100)가 주어지고, 다음에는 n개의 수가 주어진다. Input으로 주어지는 수는 1,000,000을 넘지 않는다.</p>

### Output 

 <p>각 테스트 케이스마다 가능한 모든 쌍의 GCD의 합을 Output한다.</p>
