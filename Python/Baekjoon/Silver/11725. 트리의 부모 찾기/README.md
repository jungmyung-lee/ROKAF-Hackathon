### Problem Description

<p>루트 없는 Tree가 주어진다. 이때, Tree의 루트를 1이라고 정했을 때, 각 노드의 부모를 구하는 프로그램을 작성하시오.</p>

### Input 

 <p>첫째 줄에 노드의 개수 N (2 ≤ N ≤ 100,000)이 주어진다. 둘째 줄부터 N-1개의 줄에 Tree 상에서 연결된 두 정점이 주어진다.</p>

### Output 

 <p>첫째 줄부터 N-1개의 줄에 각 노드의 부모 노드 번호를 2번 노드부터 순서대로 Output한다.</p>
