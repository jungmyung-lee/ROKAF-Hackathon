### Problem Description

<p>정수를 저장하는 Stack을 Implementation한 다음, Input으로 주어지는 명령을 처리하는 프로그램을 작성하시오.</p>

<p>명령은 총 다섯 가지이다.</p>

<ul>
	<li>push X: 정수 X를 Stack에 넣는 연산이다.</li>
	<li>pop: Stack에서 가장 위에 있는 정수를 빼고, 그 수를 Output한다. 만약 Stack에 들어있는 정수가 없는 경우에는 -1을 Output한다.</li>
	<li>size: Stack에 들어있는 정수의 개수를 Output한다.</li>
	<li>empty: Stack이 비어있으면 1, 아니면 0을 Output한다.</li>
	<li>top: Stack의 가장 위에 있는 정수를 Output한다. 만약 Stack에 들어있는 정수가 없는 경우에는 -1을 Output한다.</li>
</ul>

### Input 

 <p>첫째 줄에 주어지는 명령의 수 N (1 ≤ N ≤ 10,000)이 주어진다. 둘째 줄부터 N개의 줄에는 명령이 하나씩 주어진다. 주어지는 정수는 1보다 크거나 같고, 100,000보다 작거나 같다. 문제에 나와있지 않은 명령이 주어지는 경우는 없다.</p>

### Output 

 <p>Output해야하는 명령이 주어질 때마다, 한 줄에 하나씩 Output한다.</p>
